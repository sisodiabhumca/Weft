# Weft Terminal UI Images

This directory contains screenshots and UI mockups for Weft Terminal.

## Images

- `main-interface.png` - Main terminal interface with AI panel
- `ai-suggestions.png` - AI command suggestions in action
- `collaboration.png` - Real-time collaboration session
- `themes.png` - Theme customization interface
- `plugins.png` - Plugin management interface
- `debugging.png` - Performance monitoring and debugging tools

## Adding New Images

When adding new screenshots:
1. Use PNG format for best quality
2. Maintain consistent aspect ratio (16:9 recommended)
3. Include window decorations for context
4. Use high resolution (2x scaling recommended)
5. Optimize file size (use tools like pngcrush)
